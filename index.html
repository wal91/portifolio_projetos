Projeto de Análise de Dados, Html e Css

Banco Relacional, Python e SQL Para Análise de Dados

Esse projeto se da aos dados de pacientes que desenvolveram ou não diabetes. Gerei uma amostra de dados com os pacientes com mais de 50 anos e para cada um deles indiquei em uma nova coluna se o paciente está normal (índice de massa corpórea menor que 30) ou obeso (índice de massa corpórea maior ou igual a 30). Então gerei um novo arquivo CSV para encaminhar para o Cientista de Dados. Resolvi esse problema com Banco de Dados, Python e SQL. Os dados foram inicialmente carregados com Linguagem Python. Fiz uma cópia dos dados para um banco de dados usando a Linguagem SQL para as transformações necessárias. Por fim, copiei os dados transformados de volta para um dataframe do Pandas para salvar o resultado em formato CSV.
